# Indexes

Analogy:
An index is like a book index that lets you jump directly to a topic without reading every page.

What it is:
An index is a data structure that helps the database find rows faster.

Why it helps:
Indexes speed up read queries by avoiding full collection scans.

Trade-off:
Indexes improve read performance but add overhead on writes.

Project connection:
Used in PromptPulse for search and pagination queries.
